VectorUtil

A library of useful vector functions.

Types​

Vector​

</>
type Vector = Vector3 | Vector2

Plane​

</>
type Plane = {number}

Data type representing a plane. The plane is represented by a table with 4 values. Typically used for plane intersection calculations.
